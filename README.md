> 📬 **UniMail** is an Android app that cuts through email clutter — it prioritizes your inbox, summarizes long emails, and helps you manage your schedule, all in one place.

## ✨ Features

- 🔥 **Email Prioritization** — Automatically sorts your Gmail inbox by importance so you always see what matters first
- 📝 **Email Summarization** — Get quick summaries of long emails without reading every word
- 📅 **Schedule Management** — Manage your time and tasks directly linked to your emails
- 🔗 **Gmail Integration** — Connects seamlessly with your Gmail account

## 🚀 Getting Started

### Prerequisites
- Android Studio installed
- A Gmail account
- Android device or emulator (API 24+)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/ShahtabSaif/UniMail.git
   ```

2. **Open in Android Studio**
   - Open Android Studio → `File` → `Open` → select the `UniMail` folder

3. **Set up Gmail API**
   - Go to [Google Cloud Console](https://console.cloud.google.com/)
   - Create a project and enable the **Gmail API**
   - Download `credentials.json` and place it in the app directory

4. **Build & Run**
   - Connect your Android device or start an emulator
   - Click **Run ▶** in Android Studio
